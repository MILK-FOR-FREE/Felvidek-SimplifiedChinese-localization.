# 关于这版新汉化的几句心里话：我们为什么需要一个“粗糙”的版本？

大家好。先前流传的机翻版本，大家可能已经读了很久，也习惯了那种半文半白、略带“译制腔”的调子。但我这版重译在风格上和它差别很大，甚至可以说有些“粗鄙”。

跨度这么大，大家一开始可能会不适应。所以我觉得有必要和大家聊聊，在翻译的过程中我都看到了什么，以及为什么我会做出这些选择。

### 一、 放弃“半文半白”，找回原汁原味的泥土感

很多人觉得之前那版半文半白读着挺优雅、有年代感。我不否认那种腔调本身很好看，但**好看和对不对，是两件事**。

其实，这游戏的斯洛伐克语原文**根本不文雅**。绝大多数角色说的是东斯洛伐克方言，用词粗糙、口语化、脏话连篇。主角本人尤其如此。在原文里，语言本身携带着关键情报：谁是本地人、谁是外来者，全靠口音区分；主角的口音甚至会随着他心虚、酗酒或崩溃而加重——**语言本身就是他的状态仪表。**

那为什么英文版会那么“文雅”？
因为英语的方言全都自带具体的地理坐标（比如苏格兰腔、约克郡腔）。如果用方言，就等于把中欧的村子硬生生搬到了英国。为了止损，英译者被迫放弃了阶层区分，改用 `thou / thee / dost` 这套仿古腔统一承载“异世界感”。结果就是，英文版把仿古腔铺给了所有人——女仆、士兵、村妇、主角，人人一样古雅。**语言从情报降级成了装饰。**

而中文版机翻是从英文转译的，这就导致我们把英语因为“缺零件”才做的妥协，连同短板一起进口了过来。

但中文其实不需要做这个取舍。汉语的语域轴是垂直的：“咋办”和“怎么办”、“娘的”和“天哪”——它们没有东西南北的区别，只有粗细、泥土与书卷的落差。我们完全可以用句子长度、用词粗细把这种阶层和情绪补回来。一个说话跟农夫一样糙的领主，和一个落魄酗酒的老兵，就该有他们真实的样子。**一个骂街的人如果骂得文雅，那就不是他在骂了。**

### 二、 最大的误读：他根本不是“堂吉诃德”

这可能是先前英文版带来的最大偏差——很多人把主角当成了堂吉诃德式的、活在自己幻想里的滑稽骑士。但仔细推敲，你会发现完全套不上。

堂吉诃德的内核是认知错乱，他主动把风车看成巨人，喜剧来自妄想与现实的落差。但本作的主角**没有任何错觉，他清楚地知道世界是什么样子**。
他是两场惨败的幸存者，刀架过脖子，靠扒下死人的衣服才逃出来。他偶尔端起庄重的骑士腔调，不是因为他在幻觉中，而是因为他知道“那套话管用”——为了办成事，为了把对方架住，他是在表演。当他真正交心或崩溃时，他的语言会全面降格成平白甚至破碎的短句。

堂吉诃德的落差在于“感知与现实”；而主角的落差在于“过去与现在”。他曾经真的够格说那些庄重的话，只是现在他撑不起来了。**那不是妄想，那是衰败。** 他的笑点和他的悲剧，根源是同一样东西。

更致命的是反讽的位置。堂吉诃德是那个发疯的人，而读者和桑丘是清醒的旁观者。但在这部游戏里，真正被蒙在鼓里的不是主角一个人，而是村子里的每一个人。这里没有桑丘，所有人都以自己的方式错着，唯一知情的只有屏幕外的玩家。
而且，这里的恐怖是真的。怪物是真的，阴谋是真的，人真的会死。**堂吉诃德捅的是风车，但在这里，你捅的东西会捅回来。**

这是一部粗粝的中欧乡村黑色喜剧，笑点来自处境，来自一群各有各算计、从上到下没一个正经人的村民。那层“雅致的骑士戏仿”滤镜，是翻译链条经过三道过滤后自己长出来的东西，跟原作没关系。

### 三、 逐句校对：回到斯洛伐克语原典

最后聊聊具体的翻译工作。这版汉化是直接**逐句对照斯洛伐克语原文**做的。

在这个过程中，我发现英文本地化存在不少系统性问题：地名张冠李戴、关键动词译反导致整个笑话塌掉，以及最隐蔽的“信息删除”。英文版经常会比原文短一截，漏掉导航指令、因果链条，或者一个没说完的省略号。这些问题在以往的机翻中被原样继承甚至放大。

在这次重译中，凡是发现这类问题的，我都回到了斯洛伐克语原文核实并逐条修复。

当然，半文半白和译制腔我并没有完全扔进垃圾桶。在这版里，它们被放回了该用的地方：宗教经文、正式宣誓、碑文卷轴，或是原文真正出现外语的时候。当它只作为调味品出现时，才能精准指向那种神圣感或陌生感。

### 写在最后

翻译归根结底要还原的是“效果”，而不是“手段”。

这版重译的目标非常明确：我希望中文读者体验到的，尽量接近一个斯洛伐克玩家玩到这款游戏时的真实感受——**看到一群说话粗鲁、满嘴脏话、但活生生的人，在一个荒诞的黑色喜剧里挣扎。**

如果你习惯了先前的调子，起初或许会有些别扭。但我真心相信，这些人物脱下那件不属于他们的衣服、露出本来的面目后，远比那层雅致的滤镜要有意思得多。

感谢大家的阅读与支持，祝大家游玩愉快。
