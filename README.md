## 安装与卸载方法

> **重要提示**
> * **手动切换语言：** 由于本补丁按照官方中文标准制作，安装完成后需要在游戏内手动切换语言。
> * **旧版残留警告：** 如果你曾经游玩过，或目前仍安装着其他人发放的汉化补丁，**请务必严格遵循【情况 B】的步骤操作**，否则会导致游戏报错！

### 第一步：下载补丁
前往本仓库右侧的 [Releases](在此处替换为你的Releases页面链接) 页面，下载与你游戏版本对应的汉化压缩包。

---
---

### 第二步：安装补丁（请根据自身情况选择）

#### 【情况 A】首次游玩，或**从未**使用过其他汉化补丁
1. **定位目录：** 在 Steam 库中右键点击 `Felvidek` -> 选择 `已安装文件` -> 点击 `浏览`。
2. **置入文件：** 解压下载好的压缩包，将里面的 `www` 文件夹整体拖入游戏根目录中，并选择**“替换目标中的文件”**。
3. **进入游戏：** 启动游戏，在主菜单选择 `Options`，点击 `Language` 一次，即可切换至**简体中文**。

#### 【情况 B】以前玩过，或**目前仍安装着**其他汉化补丁
1. **清理旧版汉化：** 在 Steam 库中右键点击 `Felvidek` -> 选择 `已安装文件` -> 点击 **`验证游戏文件的完整性`**，等待 Steam 修复原版文件。
2. **初始化数据：** 启动一次游戏，进入主菜单后正常退出。
3. **安装新版汉化：** 随后，请完全按照上方【情况 A】的 3 个步骤（定位目录 -> 置入文件 -> 进入游戏并切换语言）进行操作即可。

---

### 如何卸载汉化
如果你想移除汉化并恢复官方原版，操作非常简单：
* 在 Steam 库中右键点击 `Felvidek` -> 选择 `已安装文件` -> 点击 **`验证游戏文件的完整性`**。Steam 会自动重新下载并覆盖被替换的文件。
